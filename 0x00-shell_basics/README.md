pwd - prints the absolute path of the current working directory
ls - display the content list of the current directory
cd - changes the working directoy to the users home directory
ls -l - displays the working directory content in long format
ls -la - display current working directory content including hidden files
ls -a - display all files in the current and parent and root directory
mkdir - creates a directory
mv - moves a file from one directory to another
rm - deletes file
rmdir - deletes directoy
cd - - changes working directory to the previous one
lp - prints the filetype
ln - creates a symbolic link
cp - copies files
